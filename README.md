# base
android 简单的架构封装，相对于 base 而言，多了状态页

# Quick Setup

Step 1. Add the JitPack repository to your build file
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
Step 2. Add the dependency
```
dependencies {
    implementation 'com.github.dingshaoshuai888:base-ext:1.0'
}
```
